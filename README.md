# Arduino GPS SD Logger
Logging gps data with an Arduino UNO / $8 GPS Module / SD Shield with Ethernet

The GPS is connected with software serial over pins 2/3
**Receive: 2
Transmit: 3**

Uses the TinyGPS++ library.
